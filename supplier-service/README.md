# Supplier Microservice

- **Port:** 8085
- **Technology:** Python + FastAPI + SQLite
- **Purpose:** Supplier management and procurement coordination

## Key Features

- Create, read, update, and delete supplier records
- Manage supplier contact information
- Track supplier status (active/inactive)
- View all suppliers
- Record and update supplier details (company name, address, products supplied)
- Support supplier onboarding and removal
